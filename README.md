# Hola Mundo con Flask y MySQL (Docker Compose)

Servicios:

- `app`: Aplicación Flask en `src/app.py` que se conecta a MySQL.
- `db`: MySQL 8.0 con base de datos `devdb` y usuario `devuser`.

Instrucciones rápidas:

```powershell
docker-compose up --build
```

La app queda expuesta en http://localhost:5000
